# 비교연산자
### == , != , < , > , <= , >=
* == (같다), != (다르다), < (크다), > ()작다, <= (크거나 같다), >= (작거나 같다)
* 비교 연산자의 결과는 boolean이다.
  ```java
  public class java
  {
    public static void main(Sting[] avgr)
    {
    int i = 10; // = 대입연산자 
    int j = 10;    

    // i 와 j 가 같은지 비교 하는 비교 연산자           
    System.out.println(i == j ) 
    System.out.println(i == j ) 
    System.out.println(i != j );
    System.out.println(i < j);
    System.out.println( i <= j);
    System.out.println(i > j);
    System.out.println(i >= j);
    }
  }
  ```
